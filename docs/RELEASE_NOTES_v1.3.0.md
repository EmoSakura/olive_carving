# 榄雕云艺 v1.3.0

本次版本把项目从“产品化雏形”继续推进到更像正式演示产品的阶段，重点升级了工艺体验、画布创作、作品集展示与商务产品能力。

## 主要更新

- 重构工艺解构模块，`选核 / 构思 / 粗雕 / 精雕 / 抛光` 五步拥有独立内容、判断点、误区与产出说明
- 指尖互动升级为画布工作台，支持图层、文字图层、素材贴图、本地图片导入、项目草稿、项目文件导入导出与 PNG 导出
- 新增作品集独立入口与全屏演示模式
- 首页与作品集加入更强的分段入场、共享元素与展示编排
- 服务中心升级为更商务化的咨询台，支持商务诊断、向导式录入与摘要导出
- 管理后台升级为轻量 CRM，支持线索漏斗、负责人、方案标题、预计金额、提醒和跟进记录
- 修复 Windows 下数字展馆部分图片显示不稳定的问题，统一展馆主图为更稳妥的 PNG 资源

## 适用场景

- 毕业设计答辩
- 商务提案与客户演示
- 地方文旅数字方案展示
- 研学课程与展陈互动原型

## 安装包

- Windows Release：`olive_carving.exe`
- Android Release APK：通过 GitHub Actions 标签构建生成
