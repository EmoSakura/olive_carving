# 榄雕云艺 v1.3.1

本次版本聚焦于展陈浏览体验与发布链路稳定性，属于对 `v1.3.0` 的一次体验打磨和发版补强。

## 主要更新

- 首页和数字展馆的精选路线卡片补充封面图，让策展路线更直观
- 优化精选路线、大师人物卡片和时间轴卡片的布局密度与文本截断表现
- 调整互动工作台底部工具 dock 的布局方式，提升窄区域下的适配稳定性
- 更新 Android GitHub Actions release workflow 依赖版本，降低自动发版失败风险
- 补充 Windows 桌面端安装包与便携包发布能力，方便课堂演示和本地安装
- 同步更新 README、发布指南与版本号，统一到 `v1.3.1`

## 适用场景

- 毕业设计答辩演示
- 课程展示与中期汇报
- 客户试用安装包分发
- 非遗数字展陈产品迭代演示

## 安装包

- Android Release APK：通过 GitHub Actions 标签构建生成
- Windows Installer：`olive_carving_setup_v1.3.1.exe`
- Windows Portable ZIP：`olive_carving-windows-portable-v1.3.1.zip`
