# 榄雕云艺 v1.2.0

本次版本将项目从课程展示型原型升级为更接近正式产品的应用雏形。

## 主要更新

- 新增登录 / 注册页，支持真实账号认证
- 接入 Supabase，实现注册、登录与会话恢复
- 新增产品首页，提供欢迎信息、核心指标、首页精选与最近浏览
- 新增管理员后台，支持展品发布状态、首页精选与后台标记管理
- 优化数字展馆、工艺解构与指尖互动的整体流程
- 保留收藏、最近浏览、学习进度与互动作品归档等本地状态沉淀能力
- 同步更新 README、Supabase 接入文档与项目说明材料

## 适用场景

- 毕业设计答辩
- 课程展示
- 客户演示
- 非遗数字产品雏形展示

## 安装包

- Android Release APK：`app-release.apk`
